# 🛒 Dark Pattern Detection in E-Commerce Using Machine Learning

## 📌 Overview

This project focuses on detecting deceptive user interface designs, known as **Dark Patterns**, used in e-commerce websites. The project analyzes webpage content and identifies manipulative practices such as hidden costs, false urgency, forced continuity, and misdirection using Python, SQL, and Power BI.

The main objective of this project is to create awareness among users and promote ethical design practices in online shopping platforms.

---

## 🎯 Objectives

* Detect dark patterns in e-commerce websites.
* Classify different categories of manipulative designs.
* Perform data analysis using Python and SQL.
* Visualize insights using Power BI dashboards.
* Promote transparency and ethical user experience.

---

## 📊 Dataset Information

The dataset contains information related to:

* Page Text
* Urgency Words
* Popup Count
* Hidden Charges
* Price Difference
* Pattern Type
* Pattern Label

The dataset was preprocessed by removing duplicates and handling missing values before analysis.
